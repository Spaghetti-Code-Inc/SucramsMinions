This is a 2 player game, in which players compete in various minigames. 
The format was designed for chromebooks, and should be resized to fit your screen using CTRL+ or CTRL- accordingly. 

To run the game, double click CLASS-CLASH.html.

Controls for player 1 is WASD and V as the special key. For player 2 it is arrow keys and M for special key.

The code is editable if you would like to make tweaks to the minigames or even add your own.

Enjoy!
